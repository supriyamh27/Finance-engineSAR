## 🎯 Key Calculators

### Loan Calculator
```
Monthly Payment = (Amount × Rate) / (1 - (1 + Rate)^-Years)
```
Shows: Monthly payment, total interest, total amount to pay

### Investment Growth Calculator
```
Future Value = (Principal + Contribution) × (1 + Rate)^Years
```
Displays: Future value, total invested, visualization chart

### Mortgage Calculator
```
Monthly Payment = (Loan Amount × Rate) / (1 - (1 + Rate)^-Months)
Remaining Balance = Previous Balance - Principal Payment
```
Shows: Monthly payment, total interest, amortization schedule
